#  🧑🏽‍💻 Renan Paulino

**`Desenvolvedor Front-End`**

Sou um desenvolvedor em formação com foco em **front-end**, apaixonado por criar interfaces **modernas**, **funcionais** e **bem projetadas**. Tenho experiência com **HTML, CSS, JavaScript e React**, além de grande interesse por **UI/UX design**, **acessibilidade** e **experiências digitais impactantes**.

Busco constantemente referências visuais e tendências do mercado para desenvolver soluções que unam **estética**, **usabilidade** e **inovação**. Para mim, a programação é o meio — o que realmente importa é o impacto gerado em quem usa.
